# Хостел «Комфорт» - Одностраничный сайт

Современный, адаптивный сайт для хостела "Комфорт" в Реутове.

## 🚀 Быстрый старт

### Локальное тестирование

1. Скачайте все файлы в одну папку
2. Откройте `index.html` в браузере (двойной клик)
3. Или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (http-server)
npx http-server

# PHP
php -S localhost:8000
```

Затем откройте `http://localhost:8000` в браузере.

### Публикация на хостинг

#### Вариант 1: Через FTP (cPanel, Plesk)

1. Подключитесь к FTP-серверу через FileZilla или другой FTP-клиент
2. Загрузите все файлы в папку `public_html` или `www`
3. Убедитесь, что `index.html` находится в корне

#### Вариант 2: GitHub Pages (бесплатный хостинг)

1. Создайте репозиторий на GitHub
2. Загрузите файлы в репозиторий
3. Перейдите в Settings → Pages
4. Выберите ветку `main` как источник
5. Сайт будет доступен по адресу: `https://ваш-username.github.io/имя-репозитория`

#### Вариант 3: Netlify (бесплатный хостинг)

1. Перейдите на https://netlify.com
2. Перетащите папку с файлами в окно Netlify
3. Сайт автоматически развернется

#### Вариант 4: Самостоятельный сервер

1. Загрузите файлы на сервер через FTP, SFTP или SSH
2. Убедитесь, что веб-сервер (Apache, Nginx) настроен
3. Установите SSL-сертификат (Let's Encrypt)

## 📊 Аналитика

Для добавления Google Analytics, вставьте перед закрывающим `</head>`:

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
```

Замените `GA_MEASUREMENT_ID` на ваш ID.

## 🐛 Устранение проблем

### Сайт не загружается
- Проверьте, что все файлы находятся в одной папке
- Убедитесь, что браузер не блокирует локальный доступ
- Используйте локальный сервер (см. выше)

### Неправильные стили
- Очистите кэш браузера (Ctrl+F5)
- Проверьте, что `styles.css` находится в одной папке с `index.html`
- Убедитесь, что имена файлов совпадают

### Ссылки не работают
- Проверьте, что якорьки (id) написаны правильно
- Убедитесь, что JavaScript включен в браузере
