## ⚙️ ЕСЛИ ЛОГОТИП НЕ ПОКАЗЫВАЕТСЯ

**Проблема:** Логотип не появляется на странице

**Решения:**

1. **Проверьте имя файла:**
   - Файл должен называться: `logo.jpg` (точно так!)
   - Не `Logo.jpg` или `LOGO.jpg`
   - Регистр букв ВАЖЕН на некоторых серверах

2. **Проверьте расположение:**
   - Файл `logo.jpg` должен быть в корне папки
   - Там же где `index.html`
   - Не в подпапке

3. **Проверьте формат:**
   - Файл должен быть JPEG или PNG
   - Не BMP, не TIFF
   - Проверьте расширение (.jpg, не .jpeg)

4. **Проверьте размер файла:**
   - Файл не должен быть более 500 KB
   - Сейчас 54 KB - это отлично

5. **Очистите кэш браузера:**
   - Нажмите Ctrl+Shift+Delete
   - Выберите "Картинки и файлы"
   - Нажмите "Очистить"
   - Перезагрузите страницу

6. **Посмотрите консоль (DevTools):**
   - Нажмите F12
   - Вкладка "Console"
   - Ищите красные ошибки о логотипе
   - Скопируйте ошибку и гугли

## 🚀 ДАЛЬНЕЙШЕЕ УЛУЧШЕНИЕ

Если хотите оптимизировать логотип:

### Сжимаем логотип
Используйте сайты для сжатия:
- https://tinyjpg.com (сжатие JPEG)
- https://tinypng.com (сжатие PNG)
- https://imageoptim.com (Mac)

Это может уменьшить размер с 54 KB до 20-30 KB!

### Конвертируем в WebP
WebP формат еще меньше:
- Размер: 10-15 KB вместо 54 KB
- Качество: такое же хорошее
- Поддержка: все современные браузеры

Команда для конвертации:
```bash
cwebp logo.jpg -o logo.webp
```

Затем в HTML измените:
```html
<img src="logo.webp" alt="Хостел Комфорт" class="header-logo">
```
