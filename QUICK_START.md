## 🎯 За 5 минут на онлайн

### Способ 1: Netlify (Самый простой!)
```
1. Перейди на https://netlify.com
2. Перетащи папку с файлами в окно
3. ГОТОВо! 🎉
   Твой сайт: https://random-name-123.netlify.app
```

### Способ 2: GitHub Pages (Бесплатный)
```
1. GitHub → New Repository
2. Upload все файлы
3. Settings → Pages → main branch
4. Готово!
   Твой сайт: https://твое-имя.github.io/имя-репо
```

### Способ 3: FTP на хостинг (Если у тебя уже есть)
```
1. FileZilla → вводишь данные FTP
2. Перетаскиваешь файлы в public_html
3. Готово!
   Твой сайт: https://твой-домен.ru
```

