## 📝 КАК ИЗМЕНИТЬ ЛОГОТИП

### Если хотите использовать другое изображение:

1. **Подготовьте новое изображение:**
   - Формат: JPEG, PNG или WebP
   - Размер: минимум 200x200px
   - Размер файла: до 100 KB

2. **Замените файл:**
   - Найдите `logo.jpg` в проекте
   - Удалите его
   - Положите ваше новое изображение
   - Назовите его `logo.jpg` (имя ВАЖНО!)

3. **Готово!** Обновите страницу в браузере

### Если хотите изменить размер логотипа:

Отредактируйте `styles.css`, найдите:
```css
.header-logo {
  max-width: 200px;  /* ← измените это число */
}
```

Примеры:
- `max-width: 150px` - меньше
- `max-width: 250px` - больше
- `max-width: 300px` - намного больше

Потом обновите в медиа-запросах:
```css
/* Для планшетов */
.header-logo {
  max-width: 150px;  /* было 150px, измените в тот же размер пропорционально */
}

/* Для мобильных */
.header-logo {
  max-width: 120px;  /* было 120px */
}
```

## 📞 НАПОМИНАНИЕ ДЛЯ ПУБЛИКАЦИИ

**⚠️ ВАЖНО:** При загрузке сайта на интернет:

1. **Загружайте ВСЕ файлы** включая `logo.jpg`!
2. **Все в одной папке** - не создавайте подпапки для логотипа
3. **index.html в корне** - логотип должен быть там же
4. **Не переименовывайте** файл логотипа (должен быть `logo.jpg`)

### Правильная структура:
```
📁 папка-с-сайтом/
  ├─ index.html ✅
  ├─ styles.css ✅
  ├─ script.js ✅
  ├─ logo.jpg ✅ (НОВЫЙ!)
  ├─ manifest.json ✅
  ├─ robots.txt ✅
  ├─ sitemap.xml ✅
  └─ .htaccess ✅
```

### Неправильная структура ❌:
```
📁 папка-с-сайтом/
  ├─ index.html
  ├─ styles.css
  ├─ script.js
  └─ 📁 images/ ← НЕ ТАК!
      └─ logo.jpg
```
