## Локальное тестирование

### На Windows

#### Способ 1: Встроенный веб-сервер Python
```bash
# Python 3
cd путь\к\папке\с\сайтом
python -m http.server 8000

# Затем откройте в браузере: http://localhost:8000
```

#### Способ 2: Live Server в VS Code
1. Установите расширение "Live Server"
2. Откройте папку с файлами в VS Code
3. Кликните правой кнопкой на `index.html`
4. Выберите "Open with Live Server"

### На macOS/Linux

```bash
# Перейти в папку с сайтом
cd /путь/к/сайту

# Python 3
python3 -m http.server 8000

# PHP (если установлен)
php -S localhost:8000

# Node.js (если установлен)
npx http-server
```

Откройте в браузере: `http://localhost:8000`

## Развертывание на традиционном хостинге

### Вариант 1: Реутов.РУ / Российские хостеры

#### Через cPanel (если доступен)

1. **Подключение по FTP:**
   - Откройте FileZilla или другой FTP-клиент
   - Введите данные:
     - Хост: `ftp.ваш-домен.ru`
     - Имя: `ваше-имя-пользователя`
     - Пароль: `ваш-пароль`
     - Порт: 21

2. **Загрузка файлов:**
   - Перейдите в папку `public_html` или `www`
   - Загрузите все файлы (index.html, styles.css, script.js и остальные)
   - Убедитесь, что index.html находится в корне

3. **Проверка:**
   - Откройте браузер
   - Перейдите на https://ваш-домен.ru

#### Через веб-интерфейс cPanel

1. Войдите в cPanel
2. Откройте "File Manager"
3. Перейдите в `public_html`
4. Нажмите "Upload" и выберите файлы
5. Или используйте "Extract" для архивов

### Вариант 2: Beget (популярный в России)

1. Войдите в Beget Кабинет управления
2. Откройте "Управление файлами"
3. Перейдите в папку сайта
4. Загрузите файлы
5. Убедитесь, что установлен HTTPS

### Вариант 3: HOSTING.COM / TIMEWEB

1. Подключитесь по FTP
2. Перейдите в папку с сайтом
3. Загрузите все файлы
4. Установите права доступа (755 для папок, 644 для файлов)
5. Проверьте, что index.html в корне

## Развертывание на облачных платформах

### GitHub Pages (Бесплатный)

#### Способ 1: Через веб-интерфейс GitHub

1. **Создайте репозиторий:**
   - Перейдите на github.com
   - Нажмите "New repository"
   - Название: `hostcomfort` (или другое)
   - Выберите "Public"
   - Нажмите "Create repository"

2. **Загрузите файлы:**
   - Нажмите "Add file" → "Upload files"
   - Выберите все файлы сайта
   - Нажмите "Commit changes"

3. **Включите Pages:**
   - Откройте Settings
   - Перейдите в Pages
   - Выберите ветку "main" 
   - Нажмите "Save"

4. **Результат:**
   - Сайт будет доступен по адресу:
   - `https://ваш-username.github.io/hostcomfort`

#### Способ 2: Через Git (для опытных)

```bash
# Инициализируем репозиторий
cd путь/к/сайту
git init

# Добавляем файлы
git add .

# Первый commit
git commit -m "Initial commit"

# Добавляем удаленный репозиторий
git remote add origin https://github.com/ваш-username/hostcomfort.git

# Загружаем на GitHub
git branch -M main
git push -u origin main
```

### Netlify (Бесплатный, удобный)

#### Способ 1: Drag & Drop

1. Перейдите на netlify.com
2. Создайте аккаунт (через GitHub)
3. Перетащите папку с сайтом в окно Netlify
4. Сайт развернется автоматически
5. Вы получите случайный адрес типа: `random-name-123.netlify.app`

#### Способ 2: Через GitHub

1. Залейте файлы на GitHub (см. выше)
2. На Netlify нажмите "New site from Git"
3. Выберите GitHub
4. Выберите репозиторий
5. Нажмите Deploy

#### Способ 3: Кастомный домен на Netlify

1. Зарегистрируйте домен (Beget, Namecheap и т.д.)
2. На Netlify откройте Site settings
3. Добавьте кастомный домен
4. Обновите DNS записи (инструкции на Netlify)
5. Подождите 24-48 часов на полное распространение

### Vercel (Бесплатный)

1. Перейдите на vercel.com
2. Нажмите "New Project"
3. Выберите GitHub репозиторий
4. Нажмите Deploy
5. Сайт готов!

### Firebase Hosting (Google)

```bash
# Установка Firebase CLI
npm install -g firebase-tools

# Логин
firebase login

# Инициализация проекта
firebase init hosting

# Развертывание
firebase deploy
```

## Проверка и оптимизация

### Проверка в браузерах

Протестируйте в различных браузерах:

- **Windows:**
  - Chrome
  - Firefox
  - Edge
  - Internet Explorer 11 (если нужна совместимость)

- **macOS/iOS:**
  - Safari
  - Chrome
  - Firefox

- **Android:**
  - Chrome
  - Firefox
  - Samsung Internet

### Мобильная оптимизация

1. Откройте DevTools (F12)
2. Нажмите Ctrl+Shift+M для мобильного вида
3. Проверьте макеты для:
   - iPhone 12 (390x844)
   - iPad (768x1024)
   - Android (360x800)

### Проверка скорости

1. **Google PageSpeed Insights:**
   - https://pagespeed.web.dev
   - Введите адрес сайта
   - Проверьте результаты

2. **GTmetrix:**
   - https://gtmetrix.com
   - Введите адрес
   - Получите рекомендации

### Проверка SEO

1. **Google Search Console:**
   - https://search.google.com/search-console
   - Добавьте сайт
   - Отправьте sitemap.xml

2. **Yandex Webmaster:**
   - https://webmaster.yandex.ru
   - Добавьте сайт
   - Отправьте sitemap.xml

3. **Lighthouse (встроен в Chrome):**
   - Откройте DevTools
   - Нажмите Lighthouse
   - Проверьте SEO, Performance, Accessibility

### Добавление SSL сертификата

#### На традиционном хостинге

1. **Через cPanel:**
   - Откройте "AutoSSL"
   - Нажмите "Install SSL"
   - Сертификат установится автоматически

2. **Let's Encrypt (бесплатный):**
   - Обычно автоматически на современных хостингах

#### На облачных платформах

- **GitHub Pages:** HTTPS автоматический
- **Netlify:** HTTPS автоматический
- **Vercel:** HTTPS автоматический

### Мониторинг

Настройте мониторинг сайта:

1. **Uptime Robot (бесплатный):**

## Решение проблем

### Сайт не открывается

**Проблема:** 404 ошибка при открытии сайта
**Решение:**
1. Проверьте, что index.html находится в корне
2. Убедитесь, что все файлы загружены
3. Очистите кэш браузера (Ctrl+Shift+Delete)
4. Перезагрузитесь

### Стили не применяются

**Проблема:** Страница выглядит без стилей
**Решение:**
1. Проверьте, что styles.css в одной папке с index.html
2. Откройте DevTools (F12), вкладка Network
3. Посмотрите, загружается ли styles.css
4. Если нет - проверьте путь в index.html

### Контакты не работают

**Проблема:** Ссылка на почту или телефон не работает
**Решение:**
1. Убедитесь, что ссылки правильного формата
2. Для email используйте: `mailto:info@hostcomfort.ru`
3. Для телефона используйте: `tel:+79269257555`
